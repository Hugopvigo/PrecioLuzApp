## 🌟 ¿Qué es?

**PrecioLuzApp** es una aplicación web que muestra el **precio de la luz en España (PVPC)** en tiempo real, con un diseño espectacular inspirado en el **liquid glass de iOS 26**.

> 💡 Tres estilos de cristal, tema claro/oscuro automático, y toda la información que necesitas para ahorrar en tu factura de la luz.

---

## ✨ Características

| Feature | Descripción |
|---------|-------------|
| 🎨 **3 estilos de cristal** | Frost ❄️ · Liquid 💧 · Clear 🔮 |
| 🌓 **Tema automático** | Se adapta al modo claro/oscuro del sistema |
| 📊 **Gráfico 24h** | Visualización interactiva de precios por hora |
| 💰 **Mejor momento** | Te dice cuándo enchufar para ahorrar |

| 📱 **Diseño mobile-first** | Parece una app nativa de iOS |
| ⚡ **En directo** | Precio actual actualizado en tiempo real |

## 🚀 Uso rápido

1. **Abre** `PrecioLuzApp.html` en tu navegador
2. **Toca** el botón de tema (☀️/🌙) para cambiar entre claro, oscuro y automático
3. **Navega** entre Hoy y Mañana con la barra de tabs
4. **Explora** el gráfico tocando las barras para ver precios por hora
5. **Descubre** el mejor momento para usar electrodomésticos de alto consumo
